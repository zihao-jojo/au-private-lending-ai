# 2. Private Lending Product Types in Australia

## Caveat Loans

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Short-term business funding secured by a caveat on property |
| **Term** | 1-12 months |
| **Rate** | 0.77%-1.5% per month (9-18% p.a.) |
| **LVR** | Up to 65-75% |
| **Speed** | Approval in 2-5 days |
| **Requirements** | No/minimal income verification, no credit checks for some lenders |
| **Regulation** | Typically structured as business/commercial purpose (exempt from NCCP) |
| **Key players** | Mango Credit, Diverse Funding Solutions, Equity Lenders, Caveat Capital |

## Bridging Finance

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Short-term finance to "bridge" between property transactions |
| **Term** | 1-12 months (up to 36 months) |
| **Rate** | 5-12% p.a. |
| **LVR** | Up to 65-70% |
| **Speed** | As fast as 3-5 days |
| **Key players** | Funding.com.au ($20K-$10M), Aquamore ($200K-$7.5M), La Trobe, Assetline Capital |

## Second Mortgages

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Additional funding behind existing first mortgage |
| **Term** | 1-24 months |
| **Rate** | 8-15% p.a. |
| **LVR** | Combined LVR up to 70-80% |
| **Key players** | Equity Lenders, Mango Credit, Diverse Funding Solutions |

## Construction / Development Finance

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Funding from land acquisition through construction to completion |
| **Structure** | Progressive drawdowns against construction milestones |
| **Market gap** | Non-bank lenders filling A$50B+ funding gap as major banks retreat |
| **Key players** | Qualitas, MaxCap, La Trobe, Funding.com.au, Assetline, Wingate |

## Mezzanine Finance

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Subordinated debt between senior debt and equity, typically for property development |
| **Position** | Typically 10-20% of total project funding |
| **Rate** | 12-20%+ p.a. |
| **Key players** | Qualitas, MaxCap, Wingate, VMAX Finance |

## SMSF Lending

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Loans to Self-Managed Super Funds under Limited Recourse Borrowing Arrangements (LRBAs) |
| **Minimum** | Typically $200K-$300K |
| **LVR** | Up to 70% residential, 65% commercial |
| **Requirements** | Proper SMSF structure, single acquirable asset in bare trust |
| **Key players** | Thinktank, La Trobe, Liberty Financial |

## Corporate / Business Private Credit

| Attribute | Detail |
|-----------|--------|
| **Purpose** | Direct lending to mid-market corporates |
| **Deal size** | $10M-$500M+ |
| **Key players** | Metrics Credit Partners (345 loans in wholesale trust), MA Financial (165 investments), Ares |

## Product Comparison

```
Risk / Return Spectrum (low → high):

Corporate Credit → SMSF → Bridging → Second Mortgage → Construction → Mezzanine → Caveat
   6-8%           6-8%    5-12%      8-15%            10-15%         12-20%       9-18%
```
