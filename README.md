## Key Findings

- **AU private credit AUM: A$224B** (9% YoY growth), projected A$240-250B+ by 2026
- **76.8% of home loans** go through brokers — broker channel is king
- **Critical gaps**: No AU-native document AI, no AI underwriting for non-bank, CDR middleware missing
- **Regulatory tailwinds**: CDR expanding to non-bank lenders July 2026, CCR modernizing
- **Banks retreating**: A$50B+ funding gap in property development alone
- **Competition is low**: Most private lenders still run on Excel + email
